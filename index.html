Featured Projects

Damibu Feeds CMS

Custom Content Management Platform with AI Translation

The Challenge: We needed a sophisticated content management system that could deliver culturally-adapted healthcare information to diverse communities. The system had to support multiple languages while maintaining cultural sensitivity and accuracy.

My Solution: I architected and built a comprehensive CMS from the ground up, featuring an AI-powered translation system with human validation workflows. The platform enabled NHS articles to include culturally-specific information for communities including Polish, Islamic, and other cultural groups.

Key Technical Innovation: I led the migration from MySQL to PostgreSQL Aurora specifically to implement advanced vector search capabilities, dramatically improving content discoverability and relevance.

Technical Implementation:

  • Built serverless architecture using AWS Lambda and Aurora PostgreSQL
  • Integrated AI translation APIs with custom validation interface for translators
  • Implemented vector search for intelligent content matching
  • Created flexible content customization engine for cultural adaptation
Node.js AWS Lambda PostgreSQL Aurora Vector Search AI Translation React.js

Impact: Enabled personalized healthcare information delivery in multiple languages while maintaining quality and cultural sensitivity. The system now powers content delivery for thousands of users across diverse communities.

CATCH (Common Approach to Children's Healthcare)

NHS Partnership - Offline-First Parental Guidance App

The Challenge: The NHS needed a comprehensive mobile application to provide parental guidance from pregnancy through 5 years. The critical requirement was that the app must work completely offline, as many users might not have reliable internet access when they need healthcare information most.

My Solution: I developed a React Native application with a sophisticated offline-first architecture. The app synchronizes NHS medical articles and makes them fully accessible without internet connectivity, while integrating with our custom Damibu Feeds CMS for culturally-adapted content.

Technical Challenge Solved: Created a complex offline content synchronization system that handles thousands of medical articles, images, and multimedia content, with intelligent sync strategies and conflict resolution.

Key Features:

  • Complete offline functionality for all NHS articles and resources
  • Culturally-adapted content delivery based on user preferences
  • Intelligent sync when connectivity is available
  • Cross-platform compatibility (iOS and Android)
React Native Offline-First Architecture Data Synchronization CMS Integration Cross-Platform

Impact: Provided critical healthcare information access to parents regardless of connectivity or cultural background. The app serves thousands of families with reliable, culturally-sensitive healthcare guidance.

MyHouseOfMemories

National Museums Liverpool Partnership - Digital Dementia Support

The Challenge: National Museums Liverpool had a successful physical dementia support program called "House of Memories" that needed to be digitized to reach more families affected by dementia. The digital version had to maintain the accessibility and therapeutic value of the original program.

My Solution: I created a digital recreation of the acclaimed dementia support program, focusing heavily on accessibility and user experience design. The application needed to be intuitive for both dementia patients and their families.

Design Focus: Collaborated closely with cultural heritage experts and dementia care specialists to ensure the digital translation preserved the therapeutic and cultural value of the original program.

Technical Approach:

  • Built responsive web application using React.js
  • Implemented accessibility-first design principles
  • Created intuitive navigation for users with cognitive challenges
  • Optimized for various devices and screen sizes
React.js Accessibility Design User Experience Responsive Design Cultural Heritage

Impact: Extended the reach of vital dementia support services through digital innovation, making the program accessible to families who couldn't attend physical sessions. The application provides therapeutic value and support to dementia patients and their caregivers.
